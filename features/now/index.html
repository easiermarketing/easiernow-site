Feature

One task on screen, with four buttons.

Easier Now picks a task from everything you have put in, sizes each task to the hours you have left today, and puts the next task on screen. Done, Skip, Later, Log.

Start now

One task, not fifty

The screen holds a single task. There is no list to read through before you can start.

Sized to the hours you have

Easier Now counts the hours left around your strict commitments and picks a task that fits in them.

You are told what will not fit

If your tasks need 14 hours and you have 6 hours left, Easier Now shows you which ones will not fit today.

How it works

Easier Now picks. You can override.

Behind the one task sits every project, routine and calendar event you have added. Easier Now weighs the impact marks, the deadline you set, how long each task takes and how much time is left today. Tap Later to push a task down the queue, or Skip to drop it for today. If the day breaks, press Optimise schedule and the rest of it is worked out again.

Why one task instead of a list

A list does not start itself. Before you can work you have to read it, compare the items and pick one, and you do that again after every task you finish.

The Now view does the picking. One task is on screen, and finishing that task loads the next. Everything else is still stored; you just are not looking at it.

What is on the screen

  • One task, with the project it belongs to and the time it should take
  • Four buttons: Done, Skip, Later, Log
  • Tap Done and a box asks how long the task took, in minutes, with plus and minus buttons
  • A note when today’s tasks need more hours than today has left
  • A lateness pill if the task is running behind, and the deadline beside it, purple until it passes

When the afternoon breaks

A client call runs long and the plan behind the card is wrong. Press Optimise schedule. Strict tasks stay exactly where they are, the flexible ones are refitted into the time that is actually left, and anything that no longer fits today is named so you can move it or drop it.

Questions about this feature

I have abandoned every productivity app. Why would Easier Now be different?

Easier Now has nothing to maintain. There is no board to tidy, no weekly setup and no daily planning session. Miss a fortnight and you are handed the backlog one item at a time, with three buttons under each: skip it, do it today, or mark it done.

How does it know what it is allowed to move?

You mark each task strict or flexible. Strict means anchored to a time, and that covers more than meetings: picking the kids up, the dentist, going to sleep, feeding the cats, the window you keep for email. Everything flexible is fitted around those.
